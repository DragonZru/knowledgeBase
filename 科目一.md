# 罚款2000
* 驾驶拼装车，报废车上路，吊销执照，扣留车辆，罚款200-2000
* 超速50%，罚款200-2000元，可以并处吊销驾驶证

* 违反交通管制强行通行

# 罚款5000
* 伪造，变造机动车登记证书（登记证书，行驶证，驾驶证，号牌）拘留15日，扣留车辆，罚款2000-5000，同时会扣12分呀

# 罚款十万
* 组织，参与，实施以不正当手段获取驾驶证，并牟取经济利益的，3倍以上5倍以下，不超过10万元罚款

# 酒驾醉驾
1. 初次酒驾：暂扣6个月驾驶证，罚款1000-2000，扣12分
2. 再次酒驾：吊销执照，罚款1000-2000，拘留10天
3. 酒驾营运车辆：吊销驾照，罚款5000，5年内不得再次取得，拘留15天
4. 醉驾（私家车）：约束至酒醒，吊销驾照，+刑事责任，5年内不得再次取得
5. 醉驾（营运）：约束至酒醒，吊销驾照，+刑事责任，10年内不得再次取得+重新取得后不得驾驶运用车辆
6. 酒驾/醉驾+发生交通事故+构成犯罪：刑事责任+吊销驾照+终身不得取得
